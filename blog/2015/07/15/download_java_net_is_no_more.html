Download.java.net is no more ;)

Just a reminder!

If you still are using http://download.java.net/maven/2 or http://download.java.net/maven/1 and you install a new machine or do some spring cleaning on your local repository you will notice they now have finally disappeared.

For released artifacts just use the global maven repository! For SNAPSHOT artifacts use maven.java.net.
