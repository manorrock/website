JavaEE Tip #9 - A minimal Git server web application

If you want to host a Git server over HTTP(S) then this project is for you. It is a minimal, but yet functional implementation of Git over HTTP(S).

The sources have been archived and are no longer available.

Of course I want to thank the great job the folks at the JGit project have done so I could integrate it into this web application.
